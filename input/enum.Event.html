pub enum Event {
    Input(Input),
    Loop(Loop),
    Custom(EventId, Arc<Any + Send + Sync>),
}

Models all events.

Variants

Input(Input)

Input events.

Loop(Loop)

Events that commonly used by event loops.

Custom(EventId, Arc<Any + Send + Sync>)

Custom event.

When comparing two custom events for equality, they always return false.
